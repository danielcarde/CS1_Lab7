LabSeven: Text and Image Data Processing

A Java lab that reads local text files and practices processing word and image-style data through console logic.

Highlights
Uses image.txt and words.txt as supporting files.
Practices file reading and string handling.
Works with array-style data processing.
Keeps all logic in a single Java class.

Overview
LabSeven extends earlier file-processing practice by using multiple supporting text files. The lab demonstrates how a program can read different types of text data and process them through Java logic.
The assignment reinforces loops, strings, arrays, and file input.

Why This Project
This project helps connect beginner Java syntax to more realistic data-processing tasks. Handling multiple files prepares the ground for later labs that use larger datasets and more structured records.

Usage
Compile: Save the code as LabSeven.java and compile using javac LabSeven.java.
Run: Execute the compiled code with java LabSeven.
Keep image.txt and words.txt in the same folder as LabSeven.java.

Installation
Ensure you have the Java Development Kit (JDK) installed on your system.
Open a terminal in the lab folder before compiling so the program can find any required local text files.

Architecture
File Structure: The project consists of LabSeven.java, image.txt, and words.txt.
Components: The Java file contains the processing logic. The two text files provide the input data.

Future Improvements
Add clearer output formatting.
Allow the user to choose which file to process.
Add validation for unexpected file contents.
